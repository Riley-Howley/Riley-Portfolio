2021 - PowerSMS

A mobile java application called PowerSMS was created to monitor power in server racks and remote internet towers. There is a feature that, when the power goes out, notifies network technicians by text message which location has lost power. Since I had just recently learned flutter development on my own, this project was a fantastic learning experience. I learned a lot about state management, intents, and accessing device storage.

2022 - VTM
(In Progress)

Admins can delegate incoming jobs to technicians via a mobile and web client ticketing system. The mobile application was created using the flutter framework, which enables different users to each have their own account, receive a unique ticket assignment, and view other tickets. The primary goal of the ticketing system is to address the client's issue so they can learn how their project is progressing. This helps the technician reach the client quickly for reasons of health and safety. To safeguard technicians' safety, the office administrators are also notified at each stage of the ticket. The application is demonstrated in the video below. The creation of the online tool that office administrators will use to assign assignments to technicians is still in progress. Tracking of technicians, the status of all tickets, completed jobs, and many more functions will be included.
